步驟一：建立 Google Sheets

  1. 開啟 sheets.new 建立新表格
  2. 第一列填入標題：日期 肌群 動作 組次 重量 次數

步驟二：建立 Apps Script

  1. 在試算表中點「擴充功能」→「Apps Script」
  2. 貼上以下程式碼，儲存後部署為網頁應用程式，存取權設為「任何人」
function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var rows = JSON.parse(e.postData.contents);
  rows.forEach(function(r){ sheet.appendRow(r); });
  return ContentService.createTextOutput(
    JSON.stringify({status:'ok', added:rows.length})
  ).setMimeType(ContentService.MimeType.JSON);
}

function doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var rows = sheet.getDataRange().getValues();
  rows.shift();
  return ContentService.createTextOutput(
    JSON.stringify({status:'ok', rows:rows})
  ).setMimeType(ContentService.MimeType.JSON);
}

已部署過請點「部署」→「管理部署作業」→ 編輯 → 版本選「新版本」→ 部署，URL 不變。

下載：Sheets → 手機

只匯入手機內還沒有的紀錄，不覆蓋現有資料。

知識庫（Google Docs）

將信任的訓練資料整理成 Google Doc，設為「知道連結的人都可以檢視」後貼上連結。每次 AI 分析時自動讀取最新內容。

使用說明
  1. 到 console.anthropic.com 註冊並取得 API Key
  2. 加值最低 $5 美元（約台幣 160 元）
  3. 建立 Google Doc，整理你信任的訓練知識
  4. 將 Doc 設為「知道連結的人都可以檢視」
  5. 貼上連結並儲存，在分析頁面點「開始 AI 分析」即可
